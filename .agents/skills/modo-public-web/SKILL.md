# Modo Public Web

- Priorizar velocidad móvil.
- Catálogo fácil de explorar.
- Registro no debe bloquear compra salvo decisión explícita.
- Proteger datos personales.
- Permitir eliminar cuenta solo según reglas del negocio.
- Evitar duplicación de pedido.

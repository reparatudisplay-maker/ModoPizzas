# Dirección visual

## Objetivo

Interfaz administrativa sencilla, rápida y agradable para restaurante.

## Estilo

- fondos neutros o cálidos;
- rojo como acento;
- bordes discretos;
- sombras mínimas;
- tarjetas y tablas claras;
- jerarquía tipográfica simple;
- sin exceso de efectos.

## Navegación prioritaria

1. Nuevo pedido
2. Pedidos
3. Cocina
4. Domicilios
5. Caja
6. Inventario
7. Compras
8. Gastos
9. Reportes
10. Configuración

## Reglas

- No esconder acciones frecuentes.
- No usar controles pequeños.
- No depender solo del color.
- En móvil, transformar tablas complejas.
- Mantener visible el total del pedido cuando sea posible.

# Plantilla: crear módulo

Usa esta secuencia:

1. Inspecciona módulos similares.
2. Resume lo que ya existe.
3. Propón plan de 3 a 7 pasos.
4. Identifica cambios de:
   - interfaz;
   - servidor;
   - base de datos;
   - seguridad;
   - pruebas.
5. Implementa en incrementos pequeños.
6. Verifica.
7. Resume por archivo.

No hagas commit, push, deploy ni migraciones remotas sin autorización.
